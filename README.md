## ルール

- ゲームの操作は takanamito が担当
- 新たな街やダンジョンに入るたびに Claude Code に攻略法を相談
- パーティ編成・技構成・バトル戦略は Claude Code が考案し、takanamito がそれに従う
- エリア内の移動やバトル操作は takanamito が行う

## 記録方法

- takanamito が「レポート」と言ったら、攻略ログをマークダウンで保存
- 攻略ログには戦略・パーティ編成（Lv含む）・技構成を記録
- スクリーンショットも随時保存し、記録に含める
