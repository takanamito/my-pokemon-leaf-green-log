## 準備しておくこと（前章までに）

## 攻略概要

- **対象ジム**: トキワシティジム（アースバッジ）
- **ジムリーダー**: サカキ（じめんタイプ）
- **エリア範囲**: グレンじま → 1〜3の島ナナシマ → きのみの森 → マサラ → トキワジム
- **推奨レベル目安**: 主力 Lv37〜49
- **対応レポート**: レポート 025

## 攻略のコツ

- **サカキ戦はみずタイプ高火力技で完封しやすい**。サイホーン・ニドキングは水4倍、ダグトリオ・ニドクインも水2倍弱点
- **2の島「かわのみさき」のじいさんから、御三家最終進化専用技を伝授**してもらえる（先頭の御三家最終進化に応じて: カメックス → ハイドロカノン / フシギバナ → ハードプラント / リザードン → ブラストバーン、いずれも威力150）
- **トキワジムでサカキから わざマシン26 じしん入手** → ニドキングのあなをほると入れ替えで完成形に
- **ゲームコーナーで わざマシン24 10まんボルト**を入手しておくと、ニドキングのでんげきは（威力60）→10まんボルト（威力95）に上位互換更新
- **マサキの誘いに乗らない選択肢もある**が、ナナシマ攻略でつきのいし、御三家専用技、図鑑大量追加が見込めるので進む価値あり
- **サカキ戦前にナッシーにおまもりこばん装備＋先頭にして賞金2倍**（後の四天王資金集めに有効）

## 攻略ルート

1. **グレンじま**でマサキの誘いに乗船 → **トライパス**入手 → 1〜3の島が船で行き来可能に
2. **1の島（むすびじま）** — ニシキと出会う、マサキから**いんせき**預かる
3. **2の島（ふたごじま）** — マヨちゃんの父親が娘の捜索を依頼
   - **かわのみさき** — じいさんに**御三家最終進化を先頭にして話しかける** → 御三家別の専用技伝授（ハイドロカノン / ハードプラント / ブラストバーン、威力150）

   ![ゼニガメ系列の専用技ハイドロカノン習得例](../images/024_sevii_islands/024_24_blastoise_learned_hydro_cannon.jpeg)

   - ワザマニア（小さなキノコ等で技習得）、ゲームコーナー
4. **3の島** — 暴走族「カントー連合」3連戦＋ボス戦（マタドガスのじばく注意）
5. **きずなばし → きのみの森**
   - 道中ふたごちゃんユイとリコのダブルバトル、アロマなおねえさんチヅル戦
6. **きのみの森探索**
   - きのみ大量回収（パイル、オレン、ナナシ、セシナ、クラボ、ブリー、ラム、キー）
   - **ピーピーマックス**多数
   - 図鑑用捕獲: スリープ、ピジョン、ウツドン
   - **マヨ救出** — 怖いポケモンの正体は**スリーパー Lv30**
7. **2の島でマヨを父に届ける** → **イアのみ**＋**つきのいし**入手

   ![マヨ救出](../images/025_berry_forest_tokiwa_gym/025_28_mayo_rescued.jpeg)

8. **1の島でマサキと再会** → クチバ港帰還、以後クチバから常時1〜3の島に船で行ける
9. **クチバ → なみのり → マサラタウン → 21番水道 → トキワシティ**（オーキドに図鑑進捗報告）
10. **タマムシゲームコーナーで わざマシン24 10まんボルト**（4000枚）→ ニドキング装備
11. **トキワジム挑戦** — じめんタイプ使いトレーナー連戦
12. **サカキ撃破 → アースバッジ＋わざマシン26 じしん**

    ![サカキ敗北](../images/025_berry_forest_tokiwa_gym/025_54_gym_leader_defeated.jpeg)

    ![わざマシン26 じしん入手](../images/025_berry_forest_tokiwa_gym/025_55_tm26_jishin_get.jpeg)

13. ニドキングに**じしん**装備（あなをほると入れ替え）→ **カントー全8バッジ達成**

   ![マサラタウン帰還](../images/025_berry_forest_tokiwa_gym/025_38_arrived_masara_town.jpeg)

### 推奨戦術

**最有効タイプ**: みず（4匹に2倍以上）。次点でくさ・こおり

| 御三家選択 | 推奨戦術 |
|-----------|---------|
| **ゼニガメ系** | **なみのり / ふぶき / ハイドロカノン**で4倍ワンパンの嵐。最も有利 |
| **フシギダネ系** | くさ技がサイホーン・ニドクイン・ニドキングに4倍／2倍。**ギガドレイン / ハードプラント**で完封ライン |
| **ヒトカゲ系** | ほのお技は等倍が多くて決め手にならない。野生の**ナッシー（くさ/エスパー）** や別ポケモンを補助に。リザードンの**つばさでうつ**でダグトリオ等倍 |

**共通の戦術**:

- 物理アタッカー（ニドキング・カビゴン）はサカキのじしんに弱いため控え推奨
- **ナッシーにおまもりこばん装備して先頭 → 即みずタイプ主力に交代**で賞金2倍を確保

### 本プレイのバトル記録（ゼニガメ選択）

- **採用パーティ**: ナッシー（おまもりこばん先頭）→ カメックス Lv49 / ニドキング Lv49 控え
- **戦況**: ニドキングがサイホーンにかわらわり開幕後、**カメックスのなみのり連発で全員撃破**
- 5匹中4匹が水4倍or2倍 → ほぼワンパン
- カメックスのなみのり一辺倒で完封できる相手
