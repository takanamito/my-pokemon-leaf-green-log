# 第5章 キョウ（セキチクシティ・どくタイプ）

> ピンクバッジ獲得まで。16番道路（カビゴン捕獲）→ サイクリングロード → セキチクシティ → サファリゾーン → なみのり・かいりき入手 → キョウ初挑戦で全滅 → ヤマブキ遠征でサイコキネシス入手 → リベンジ。
>
> 元レポート: [015 カビゴン捕獲〜ニドキング爆誕](../reports/015_route16_snorlax.md) / [016 サファリゾーン](../reports/016_safari_zone.md) / [017 育成＆図鑑40種〜キョウ全滅](../reports/017_levelup_pokedex.md) / [018 ヤマブキ遠征＆キョウリベンジ](../reports/018_revenge_sekichiku_gym.md)

## 準備しておくこと（前章までに）

- **バッジ4個**取得済み（〜レインボーバッジ）
- 主力ポケモン Lv30以上
- **ポケモンのふえ**入手済み（カビゴン起こし用）
- **シルフスコープ**入手済み
- 推奨ポケモン: 御三家最終進化、ブースター（前章でイーブイ入手済み）、マンキー（オコリザル進化済み or 進化目前）、ゴース（前章ポケモンタワーで捕獲）
- 推奨アイテム: いいキズぐすり大量、どくけし、まひなおし、いいつりざお／ボロのつりざお

## 攻略概要

- **対象ジム**: セキチクシティジム（ピンクバッジ）
- **ジムリーダー**: キョウ（どくタイプ）
- **エリア範囲**: タマムシ西 → 16/17/18番道路 → セキチク → サファリゾーン → 12/13/14/15番道路 → セキチクジム
- **推奨レベル目安**: 主力 Lv30〜43
- **対応レポート**: レポート 018（リベンジ成功）

## 攻略のコツ

- **キョウ戦の最大の壁は「ベトベトン Lv39」**。「ちいさくなる + とける + すごいきずぐすり」の三重コンボで、物理アタッカーは詰む。**エスパー特殊技で速攻撃破が必須**
- **ヤマブキシティのエスパーおやじから わざマシン29「サイコキネシス」を取りに行く**。タマタマ（サファリゾーン）→ リーフのいしでナッシー → サイコキネシス習得が、ベトベトン速攻ラインの正解
- **ポケモンのふえでカビゴンを起こす**（タマムシ西 16番道路）。Lv30の壁役・特殊耐久ポケモンとして即戦力。「けたぐり」は460kgの重さで一撃即死、捕獲時は使わない
- **16番道路の隠し家でひでんマシン02 そらをとぶ入手**。じんすけ（カモネギ、クチバでオニスズメと交換）に覚えさせるとひでん要員1匹で完結
- **サファリゾーンで ひでんマシン03 なみのり / 園長救出で ひでんマシン04 かいりき**を入手。両方ジム後の探索とイベント進行に必須
- **タマタマ → ナッシー進化用のリーフのいしはサファリゾーンに落ちている**。逃さず取る
- **ハイパーボールはセキチクシティから解禁**（レベル4バッジまでは購入不可）

## 攻略ルート

1. **タマムシ西 16番道路** — サイクリングロード北口
   - 隠し家で **ひでんマシン02 そらをとぶ** 入手
   - **カビゴン Lv30** をポケモンのふえで起こして捕獲（まひ→ねむる回復に注意、けたぐり厳禁）

   ![カビゴン捕獲成功！](../images/015_route16_snorlax/015_snorlax_caught.jpeg)

2. **クチバへ寄り道** — オニスズメ ↔ **じんすけ（カモネギ）** 交換、ながねぎ装備
3. **タマムシ → シオン経由 → 12〜15番道路を南下**
   - すごいつりざお入手（12番道路つりおやじの弟）
   - 道中でわざマシン27 おんがえし、わざマシン18 あまごい
   - 2匹目のカビゴン（道塞ぎ）はオコリザルのけたぐりで一撃
   - **ニドラン♂ → ニドリーノ進化 → つきのいしでニドキング爆誕**

   ![ニドキング爆誕！](../images/015_route16_snorlax/015_nidorino_evolution_to_nidoking.jpeg)

4. **セキチクシティ到着**
5. **サファリゾーン**（500円、サファリボール30個）
   - 園長の **きんのいれば** を奥地で拾い → **ひでんマシン04 かいりき**
   - **ひでんマシン03 なみのり**（トレジャーハウス）
   - リーフのいし、ふしぎなアメ、わざマシン11/47/32、タウリン
   - 図鑑用: パラセクト、タマタマ、サイホーン、トサキント、アズマオウ
   - ミニリュウは超低確率＋逃げやすいので別途リベンジ枠

   ![ひでんマシン03 なみのり入手！](../images/016_safari_zone/016_get_hm03_surf.jpeg)

6. **みずタイプ主力になみのり、ノーマル・かくとう枠（カビゴン等）にかいりき習得**
7. **キョウジム挑戦準備**: タマタマ→ナッシー進化、ヤマブキで わざマシン29 サイコキネシスをナッシーに習得
   - **空手道場**（ヤマブキ南）でカラテ大王タケノリ撃破 → サワムラー or エビワラー1匹もらえる
8. **サイクリングロードで仮想キョウ戦**（ベトベトンLv33をナッシーで2発確認）
9. **キョウジム → リベンジ成功**

   ![キョウのベトベトン撃破！](../images/018_revenge_sekichiku_gym/018_27_betbeton_defeated.jpeg)

## ジム攻略 — キョウ戦

### リーダーの手持ち

| ポケモン | Lv | 主要技 | 弱点 / 注意点 |
|---------|----|--------|---------------|
| ドガース | 37 | スモッグ / どくガス / じばく | 特性ふゆう（じめん技無効） |
| ベトベトン（エース1） | 39 | ヘドロこうげき / ちいさくなる / とける / ヘドロばくだん | 物理に強い、すごいきずぐすり持参で長期戦化 |
| ドガース | 37 | じばく注意 | ふゆう |
| マタドガス | 43 | ヘドロばくだん / だいばくはつ / すごいきずぐすり | エース、特殊耐久も高い |

### 推奨戦術

**最有効タイプ**: エスパー特殊（どくに2倍）。次点でじめん（ベトベトン系限定、ふゆう持ちには無効）

| 御三家選択 | 推奨戦術 |
|-----------|---------|
| **ゼニガメ系** | みず技は等倍、決め手にならない。**ナッシー（タマタマをサファリで捕獲→リーフのいしで進化）にサイコキネシス**を持たせ主軸に |
| **フシギダネ系** | くさ技はどく半減で不利。**ナッシーのサイコキネシス**主軸推奨。フシギバナはバックアップ |
| **ヒトカゲ系** | リザードンのほのお技は等倍。**ナッシーのサイコキネシス**主軸＋リザードンのつばさでうつ（ひこう→どく等倍）でサポート |

**共通の戦術**:

- **タマタマ（サファリゾーン）→リーフのいしでナッシー進化**を必ず行い、**わざマシン29 サイコキネシス**（ヤマブキのエスパーおやじ）を覚えさせる
- **ベトベトンが「ちいさくなる」を積む前にサイコキネシスで叩く**。物理アタッカーで挑むと回避率＋すごいきずぐすりで詰む
- **ふゆう持ち（ドガース・マタドガス）にじめん技は通らない**。じしんを撃つ場合はベトベトンに限定
- **すごいきずぐすりを最低3個、まひなおしも持参**

### 本プレイのバトル記録（ゼニガメ選択）

- **初挑戦（レポート017）**: ベトベトンに「ちいさくなる + とける + すごいきずぐすり」で全パーティ壊滅。ドガース自爆でニドキング道連れ → 全滅
- **リベンジ（レポート018）**: ナッシー Lv31 にサイコキネシス習得 → ベトベトン速攻撃破ライン確立
- 結果: ナッシーがほぼ単騎で3匹掃討、マタドガスはカメックス Lv43 のなみのりで仕留める
- ナッシー HP6/99 までいったが粘り勝ち

→ 詳細: [レポート 017](../reports/017_levelup_pokedex.md)（全滅）、[レポート 018](../reports/018_revenge_sekichiku_gym.md)（リベンジ）
