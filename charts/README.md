# 攻略チャート

殿堂入りまでの攻略を**ジム単位**で再構成したチャート集です。

> 📘 **攻略の本筋（攻略のコツ・攻略ルート・推奨戦術）は御三家を問わず使える汎用情報として書かれています**。各章「ジム攻略 — 推奨戦術」では、ゼニガメ／フシギダネ／ヒトカゲ別に選択肢を併記しています。
>
> 「本プレイのバトル記録」「本プレイのパーティ推移」セクションのみ、御三家ゼニガメ選択時の参考例として実プレイの記録を載せています。

各章は以下の構成で書かれています:

- **準備しておくこと** — 前章までに揃えておくべきポケモン・道具・推奨レベル
- **攻略概要** — 対象ジム / リーダー / エリア範囲 / 推奨レベル
- **攻略のコツ** — このエリアで詰まらないための要点
- **攻略ルート** — ストーリー進行順のステップ
- **主要トレーナー戦** — エリア内の主要トレーナーと手持ち
- **このエリアで仲間になるポケモン** — 推奨度付き
- **入手アイテム** — エリア別 + ジム撃破報酬
- **ジム攻略** — リーダー手持ち・推奨戦術・本プレイのバトル記録
- **本プレイのパーティ推移** — ゼニガメ選択時の参考例（技構成と進化トピック）
- **次の章へ** — 次の目的地と達成チェックリスト

## 用語集

ポケモン攻略に出てくる用語の簡易解説。

| 用語 | 説明 |
|------|------|
| **タイプ一致（タイプ一致ボーナス）** | 自分と同じタイプの技を使うと、威力が1.5倍になる仕組み |
| **ばつぐん（2倍弱点）** | タイプ相性で相手が弱点をついた場合、ダメージが2倍 |
| **4倍弱点** | 相手が複合タイプで両方とも弱点の場合、ダメージが4倍（例: いわ/じめんのイワークにみず技で4倍） |
| **いまひとつ** | タイプ相性が悪い場合、ダメージが半減（0.5倍） |
| **こうかなし** | タイプ相性で完全に無効（ノーマル/かくとう→ゴースト等） |
| **特性** | 各ポケモン固有の能力。ふゆう（じめん技無効）、ちょすい（みず技無効＋HP回復）など、戦況を変えるものが多い |
| **物理技 / 特殊技** | 第3世代では技ごとに分類。物理は「こうげき」依存、特殊は「とくこう」依存。打ち分けでダメージが変わる |
| **わざマシン** | 一度だけ使える技教え道具。第3世代では1つにつき1回限定なので使い所を選ぶ |
| **ひでんマシン** | フィールドでも使える技教え道具（いあいぎり、なみのり等）。複数回使用可、ジム前にバッジが必要 |
| **進化レベル / 進化アイテム** | レベルアップで進化するポケモンと、特定アイテム（つきのいし、ほのおのいし等）で進化するポケモンがいる |
| **通信交換進化** | 他プレイヤーと通信交換すると進化する種類（ゲンガー、カイリキー、ゴローニャ、フーディン）。ソロプレイ環境では入手不可 |
| **EVS / こうげきUP系道具** | タウリン（こうげき）、キトサン（ぼうぎょ）、リゾチウム（とくこう）、ブロムヘキシン（とくぼう）、マックスアップ（HP）、インドメタシン（すばやさ）。各ステータスの努力値を上げる |
| **持ち物** | ポケモンに持たせる道具。くろいメガネ（あく技1.1倍）、おまもりこばん（賞金2倍）など |

## 主要な長期動線

複数の章をまたぐ重要な収集・進化動線。

### 化石ポケモン（オムナイト or カブト）

- **第2章 オツキミヤマ**で「**かいのカセキ**」or「**こうらのカセキ**」を1つ選んで入手
- **第7章 グレンじま研究所**で復元してオムナイト or カブトを入手
- 図鑑コンプリート向け、戦力としては育成必須

### 通信交換進化（任意）

- **ゲンガー**（ゴース→ゴースト Lv25→通信交換）: 第4章でゴース捕獲、第6章でゲンガー化が理想
- **カイリキー**（ワンリキー→ゴーリキー Lv28→通信交換）: 第4章のイワヤマトンネルでワンリキー捕獲
- **ゴローニャ**（イシツブテ→ゴローン Lv25→通信交換）: 第2章オツキミヤマや第4章イワヤマでイシツブテ捕獲
- **フーディン**（ケーシィ→ユンゲラー Lv16→通信交換）: 各エリアで捕獲機会あり
- **通信できない環境では、ゴースト・ゴーリキー・ゴローン・ユンゲラー止まりでも十分戦力**

### 御三家最終進化専用技（第8章）

- **2の島かわのみさき**のじいさんから、御三家最終進化を先頭にして話しかけると専用技伝授
  - カメックス → **ハイドロカノン**（みず威力150）
  - フシギバナ → **ハードプラント**（くさ威力150）
  - リザードン → **ブラストバーン**（ほのお威力150）
- 使用後1ターン動けない反動あり、最後の一手として活用

### ひでん要員（フィールド技用ポケモン）

- 必要なひでん技: いあいぎり / そらをとぶ / なみのり / かいりき / フラッシュ / いわくだき
- **カモネギ「じんすけ」**（クチバ NPC でオニスズメと交換）が「いあいぎり」「そらをとぶ」両対応で省スペース
- なみのり・かいりきは御三家ゼニガメ系列のカメックス、カビゴンなど主力ポケモンに兼用させると枠を節約できる
- フラッシュ（パラス等の くさ系）、いわくだき（ワンリキー等のかくとう系）は専用枠が必要

### 伝説ポケモン

- **フリーザー Lv50**（第7章 ふたごじま）: ハイパーボール大量必要、最低捕獲率
- **ファイヤー Lv50**（第9章 ともしびやま、ナナシマ解放後）: 同上
- **サンダー Lv50**（無人発電所、殿堂入り後）: 同上
- **ミュウツー Lv70**（殿堂入り後、ハナダの洞窟）: 最強格

### マスターボール

- **第6章 シルフカンパニー**でサカキ撃破後、社長から1個もらえる
- 100%捕獲、伝説ポケモン用に温存推奨（フリーザー・サンダー・ファイヤー・ミュウツーのいずれかに）

### 推奨育成ポケモン（殿堂入りまで）

- **みず系御三家（カメックス）** or 同等のみずタイプ — エースアタッカー
- **ナッシー**（タマタマ→リーフのいし）— エスパー特殊（キョウ・四天王キクコで活躍）
- **ニドキング**（ニドラン♂→ニドリーノ→つきのいし）— どく/じめん多技範囲
- **カビゴン**（12 / 16番道路、ふえで起こす）— 耐久壁
- **ブースター**（タマムシマンションのイーブイ→ほのおのいし）— ほのお枠
- **ゲンガー or ゴースト**（ポケモンタワーでゴース捕獲）— ゴースト/特殊
