# 攻略チャート

殿堂入りまでの攻略を**ジム単位**で再構成したチャート集です。

> 📘 **攻略の本筋（攻略のコツ・攻略ルート・推奨戦術）は御三家を問わず使える汎用情報として書かれています**。各章「ジム攻略 — 推奨戦術」では、ゼニガメ／フシギダネ／ヒトカゲ別に選択肢を併記しています。
>
> 「本プレイのバトル記録」「本プレイのパーティ推移」セクションのみ、御三家ゼニガメ選択時の参考例として実プレイの記録を載せています。

各章は以下の構成で書かれています:

- **準備しておくこと** — 前章までに揃えておくべきポケモン・道具・推奨レベル
- **攻略概要** — 対象ジム / リーダー / エリア範囲 / 推奨レベル
- **攻略のコツ** — このエリアで詰まらないための要点
- **攻略ルート** — ストーリー進行順のステップ
- **主要トレーナー戦** — エリア内の主要トレーナーと手持ち
- **このエリアで仲間になるポケモン** — 推奨度付き
- **入手アイテム** — エリア別 + ジム撃破報酬
- **ジム攻略** — リーダー手持ち・推奨戦術・本プレイのバトル記録
- **本プレイのパーティ推移** — ゼニガメ選択時の参考例（技構成と進化トピック）
- **次の章へ** — 次の目的地と達成チェックリスト
