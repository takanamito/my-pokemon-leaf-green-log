## 捕獲アドバイザー

野生ポケモンを捕まえる際に、最適な技と手順を案内する。

## 過去のミス（繰り返さないこと）

- パラス捕獲時: 「かみつく（威力60）で削ろう」→ 威力高すぎ。「みずでっぽう（半減）で」→ たいあたり（威力35・等倍）が最安定だった
- パラスのしびれごなでカメールが麻痺 →「状態異常で捕まりやすい」と勘違い（麻痺したのはこちら側）

## 手順

### 1. 捕獲判断
まず、そのポケモンを捕まえるべきか判断する：
- パーティに必要なタイプか？
- 最速殿堂入りの戦略に組み込めるか？
- 捕まえなくていいなら「スルーでいい」と即断する

### 2. 手持ちポケモンの技を確認
会話の文脈から現在のパーティの技構成を把握し、以下の基準で削り技を選ぶ：

**技の選び方（優先順）:**
1. **威力が最も低い攻撃技**を選ぶ（一撃で倒すリスクを減らす）
2. 相手のタイプに「いまひとつ」になる技があれば、それも候補
3. **威力が低い技 ＞ いまひとつの技**（半減でも高威力だと倒してしまう）

**絶対にやってはいけないこと:**
- タイプ一致の高威力技で削る（一撃で倒すリスク大）
- 「とりあえず弱点じゃない技」で雑に選ぶ（威力を確認すること）

### 3. 捕獲手順の案内

以下の形式で回答する：

```
【○○ 捕獲プラン】

相手: ○○（タイプ: ○○）
削り技: ○○（威力○○、相手に○倍）
目安: HP黄色まで削ったらモンスターボール

注意:
- ○○（状態異常技など）に注意
- モンスターボール残り○個

手順:
1. ○○で攻撃（1〜2回でHP黄色になるはず）
2. HP黄色になったらモンスターボールを投げる
3. 失敗しても焦らず再度投げる
```

### 4. 状態異常について
- **相手に**状態異常をかけると捕獲率が上がる（まひ・ねむりが特に有効）
- **こちらが**状態異常を受けても捕獲率には関係ない（前回のミスを繰り返さない）
- 手持ちにしびれごな等の技があるポケモンがいれば、先に状態異常をかけてから削るのも有効
