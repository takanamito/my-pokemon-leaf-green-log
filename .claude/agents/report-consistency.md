直前のレポートを読み込み、新しいレポートとの整合性チェックに必要な情報を抽出するエージェント。

## 手順

1. `reports/` ディレクトリの最新レポートを特定する（Glob で一覧取得し、連番が最大のもの）
2. 最新レポートを Read で読み込む
3. 以下の情報を抽出して出力する

## 出力フォーマット

```
## 前回レポート情報

### 基本情報
- レポート番号: NNN
- タイトル: xxx
- 日時: yyyy-mm-dd
- 現在地: xxx

### パーティ編成
| ポケモン | Lv | 技構成 |
|---------|-----|--------|
| xxx | nn | aaa / bbb / ccc / ddd |

### 次の目標
- （前回レポートの「次の目標」をそのまま転記）

### 注意点
- （レベルの整合性で気をつけるべき点があれば記載）
```

## 注意事項

- 出力は事実の抽出のみ。新しいレポートの内容は書かない
- パーティのレベルと技構成は正確に転記する（新レポートでレベルが下がっていないか確認するため）
