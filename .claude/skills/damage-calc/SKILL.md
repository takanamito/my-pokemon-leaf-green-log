## 実効威力計算スキル（サブエージェント版）

バトル中の技選択で、タイプ相性の確認と実効威力の計算を **Agent ツール（subagent_type: general-purpose）に委譲する**ためのスキル。
メインの会話では計算を行わず、サブエージェントの結果をそのまま takanamito に伝える。

## いつ使うか

- バトル中に技を選ぶとき（トレーナー戦・野生戦）
- 相手のポケモンが変わったとき
- ポケモンを交代するとき

## 使い方

以下のプロンプトで Agent ツールを呼び出す。`{変数}` の部分を実際の値に置き換えること。

```
Agent ツール呼び出し:
  subagent_type: general-purpose
  description: "実効威力計算"
  prompt: 下記の内容をそのまま渡す
```

### サブエージェントに渡すプロンプト

```
ポケモンバトルの実効威力を計算してほしい。
`.claude/skills/type/SKILL.md` のタイプ相性表を必ず読んでから計算すること。推測で相性を判断してはいけない。

## 計算対象

相手ポケモン: {相手ポケモン名}（タイプ: {相手タイプ}）
※相手のタイプが不明な場合、ポケモン名から正確に特定すること。複合タイプを見落とさないこと。

味方ポケモン: {味方ポケモン名}（タイプ: {味方タイプ}）
技一覧:
- {技名1}（タイプ: {技タイプ1}、威力: {威力1}）
- {技名2}（タイプ: {技タイプ2}、威力: {威力2}）
- {技名3}（タイプ: {技タイプ3}、威力: {威力3}）
- {技名4}（タイプ: {技タイプ4}、威力: {威力4}）

## 計算ルール

1. `.claude/skills/type/SKILL.md` を読んで、各技タイプ → 相手タイプの相性倍率を確認する
2. 複合タイプの場合、各タイプの倍率を掛け合わせる（例: いわ/じめん に みず → 2.0 × 2.0 = 4.0）
3. 以下の式で実効威力を計算する:

実効威力 = 技の威力 × STAB × タイプ相性
- STAB: 技のタイプが味方ポケモンのタイプと一致 → ×1.5、不一致 → ×1.0
- タイプ相性: ばつぐん ×2.0、等倍 ×1.0、いまひとつ ×0.5、こうかなし ×0

4. タネマシンガン等の連続技は「最低回数（2回）での合計威力」で計算する

## 出力フォーマット（この形式で厳密に出力すること）

【{味方ポケモン名} → {相手ポケモン名}（{相手タイプ}）】
| 技名 | 威力 | タイプ | STAB | 相性 | 実効威力 |
|------|------|--------|------|------|----------|
| {技名} | {威力} | {タイプ} | {×1.0 or ×1.5} | {倍率と説明} | {計算結果} |

推奨技: {最も実効威力が高い技名}（実効威力: {値}）
こうかなしの技: {あれば技名を列挙、なければ「なし」}
```

## メインエージェントの動き

1. サブエージェントを呼び出す
2. 結果が返ってきたら、**推奨技をそのまま takanamito に伝える**
3. 状態異常技の判断（しびれごな等を使うかどうか）はメインエージェントが行う
4. 先頭ポケモンの判断も引き続きメインエージェントが行う（`battle/SKILL.md` の先頭ポケモン決定ルールに従う）

## 注意事項

- サブエージェントの結果を**そのまま信頼してよい**（相性表を読んで計算しているため）
- メインエージェントが自分で相性を推測して上書きしないこと
- 相手のタイプが不明な場合は、サブエージェントに特定させる（プロンプトにその旨を書く）
- 控えのポケモンの技も計算したい場合は、複数のポケモン分をまとめて1回のサブエージェント呼び出しに含めてよい
