## エリア進入前の準備チェッカー

新しいエリア（洞窟・ダンジョン・新しい道路など）に入る前に、買い物と準備の漏れがないか確認する。

## このスキルを使うタイミング（使い忘れ防止）

以下のいずれかに該当したら、**必ず**このスキルを実行する：

- 街から出発するとき
- 新しい道路・ルートに入るとき
- 洞窟・ダンジョンに入るとき
- ジムに挑戦するとき
- 長時間のバトル後にアイテムが減っているとき

**「次に進もう」と言われたら、まずこのスキルを実行する。**

## 過去のミス（繰り返さないこと）

- トキワの森: どくけしを買い忘れて全滅
- オツキミヤマ: フレンドリィショップの有無を確認せず買い物指示を出した。まひなおしの購入指示も忘れた
- ハナダシティ: このスキルの使用を忘れた。takanamito に「なんでさっき言ったときにskill使わなかったの？」と指摘された
- ハナダシティ: キズぐすりの購入を指示し忘れた。トレーニング中にアイテム不足で困った
- ナゲツリブリッジ後: ライバル戦でアイテムを消費した後、補充せずに次に進もうとした。takanamito に「アイテムは買っていかなくていいの？」と言われた

## チェック手順

### 1. 現在地の確認
- 今いる場所にフレンドリィショップがあるか確認する
- **ショップがない場合、最寄りのショップがある街を案内する**
- 洞窟・ダンジョンの入口にはショップがないことが多い。街にいるうちに買い物を済ませる

### 2. 次のエリアに出現するポケモンのタイプを考慮
- どくタイプが出る → **どくけし**を買う
- でんきタイプが出る → まひなおしを買う
- 洞窟系 → どくけし＋まひなおし両方推奨

### 3. 買い物チェックリスト
以下を takanamito に確認する：

```
□ キズぐすり（またはいいキズぐすり）: 十分な数があるか？
□ どくけし: 必要か？在庫は？
□ まひなおし: 必要か？在庫は？
□ モンスターボール: 捕まえたいポケモンがいるか？在庫は？
□ あなぬけのヒモ: 洞窟に入るなら1個は欲しい
□ 所持金: 今いくらあるか？
```

### 4. 捕まえるポケモン
- そのエリアで捕獲すべきポケモンがいるか判断する
- 判断基準:
  - パーティに足りないタイプを補えるか
  - 殿堂入りの戦略に組み込めるか
  - 進化後の性能も考慮する
- 捕まえる場合はモンスターボールの必要数も買い物リストに反映する
- 捕まえない場合は「捕まえるポケモン: なし」と明記する

### 5. 目標レベル
- そのエリアを抜けるころの目標レベルを提示する
- 判断基準:
  - エリア内のトレーナーやボスのレベル帯
  - 次のジムリーダーのポケモンのレベル
  - 進化レベルが近いなら進化を目標に含める
- 「普通に戦っていけば届く」のか「意識的にレベル上げが必要」なのかも伝える

### 6. 先頭ポケモンの決定
- `battle/SKILL.md` の「先頭ポケモン決定ルール」に従って先頭を決める
- エリアのトレーナー情報を把握し、ライバル戦やイベント戦の有無を確認する
- 育成したいポケモンがいる場合、安全に先頭にできるか判断する

### 7. パーティの確認
- HP全回復したか（ポケモンセンター）
- 状態異常が残っていないか
- PP は十分か

### 8. 回答フォーマット

```
【○○エリア 準備チェック】

買うもの:
- ○○ × N個
- ○○ × N個

捕まえるポケモン:
- ○○（タイプ: ○○ / 理由: ○○）

目標レベル:
- ○○ → Lv○○（理由: ○○）

先頭ポケモン:
- ○○（理由: ○○）
- 控え: ○○に交代する想定

やること:
- □ ポケモンセンターで回復
- □ ○○

注意点:
- ○○タイプのポケモンが出現するので○○に注意
```

## 重要

- **「たぶん大丈夫」は禁止。** 確認できないことは takanamito に聞く
- 所持金が足りない場合は、優先順位をつけて案内する
- ショップの場所を間違えない（前回のミスを繰り返さない）
