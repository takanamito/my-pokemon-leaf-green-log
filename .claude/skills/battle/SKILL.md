## バトル技選択・先頭ポケモン決定スキル

バトル中の技選択と、エリア移動時の先頭ポケモン決定を行う。

## 技推奨の前に必ず実行する3ステップ（最重要・省略禁止）

技を推奨する前に、**毎回必ず**以下の3ステップを実行すること。1回でも省略するとタイプ相性ミスが発生する。

### ステップ1: 相手のタイプを特定する

- ポケモン名から**全てのタイプ**を特定する（複合タイプを見落とさない）
- 不確かな場合は type/SKILL.md の相性表を参照する。**推測で回答しない**
- 例: バタフリー → むし/ひこう（両方のタイプを考慮する必要がある）

### ステップ2: type/SKILL.md の相性表で倍率を確認する

- 手持ちポケモンの**全攻撃技**について、技のタイプと相手のタイプの相性を確認する
- 複合タイプの場合、**両方のタイプに対する倍率を掛け合わせる**
- 「たぶんばつぐん」「たぶん等倍」は禁止。相性表で確認してから回答する

### ステップ3: 実効威力を計算して最大のものを選ぶ

```
実効威力 = 技の威力 × STAB × タイプ相性

STAB: 技のタイプ = 使うポケモンのタイプ → ×1.5、それ以外 → ×1.0
タイプ相性: ばつぐん → ×2.0、等倍 → ×1.0、いまひとつ → ×0.5、こうかなし → ×0
複合タイプ: 各タイプの倍率を掛け合わせる（例: いわ/じめん にみず → 2.0 × 2.0 = 4.0）
```

- ライバルシゲキ戦: マンキーLv10を先頭にしたままライバル戦に突入。Lv17ピジョンに何もできず交代する羽目に
- フシギダネ戦: パラスを出してねむりごなを先制で撃たれ、やどりぎ＋ねむりコンボにハマった

### レベリング戦略ミス
- 「カメールで相手のHPを削ってからパラスに交代」を指示 → takanamito に「パラスを先頭に出してからカメールに交代する方が効率的」と正された（パラスがダメージを受けない＋両方に経験値）

## 技選択ルール

### 計算例（現パーティ）

**カメール（みずタイプ）が等倍の相手を攻撃する場合:**
- みずのはどう: 威力60 × STAB1.5 × 等倍1.0 = **実効90** ← これが最強
- かみつく: 威力60 × 1.0 × 1.0 = **実効60**
- たいあたり: 威力35 × 1.0 × 1.0 = **実効35**

**パラス（むし/くさタイプ）が等倍の相手を攻撃する場合:**
- タネマシンガン: 威力25×2〜5回 × STAB1.5 = **最低実効75**（2回命中）〜**最大実効187.5**（5回命中）
- ひっかく: 威力40 × 1.0 = **実効40**

**マンキー（かくとうタイプ）がいわタイプを攻撃する場合:**
- からてチョップ: 威力50 × STAB1.5 × ばつぐん2.0 = **実効150**
- ひっかく: 威力40 × 1.0 × 1.0 = **実効40**

### 技選択の手順

1. 相手のポケモンのタイプを確認する（複合タイプも忘れずに）
2. 使えるすべての攻撃技の**実効威力を計算**する
3. **最も実効威力の高い技**を選ぶ
4. 実効威力が同じ場合は、命中率が高い方を選ぶ
5. PPが残り少ない場合は、2番目の候補を検討する

### 同じかくとうタイプの技が複数ある場合

からてチョップとけたぐりなど、同タイプの技が複数ある場合は**威力だけで比較する**。タイプ相性は同じなので、威力が高い方を選べばよい。「相性が違う」と説明しないこと。

### 状態異常技の活用判断

- 相手のレベルが高い/HP管理が厳しい → しびれごな等でまひにして安全に
- 相手のレベルが低い/余裕がある → 状態異常技は不要、攻撃で倒す
- 捕獲時はしびれごな・ねむりごなが最優先

## 技スロット管理ルール

ポケモンは**最大4つ**の技を覚えられる。

- 新しい技を覚えるとき、**まず現在いくつ技を覚えているか確認する**
- 4つ未満なら忘れさせる必要はない。空きスロットに入る
- 4つ覚えている場合のみ、どの技を忘れるか判断する
- **過去に2回同じミスをした（ゼニガメLv10のからにこもる、パラスへのTM09）。絶対に繰り返さない**

## 先頭ポケモン決定ルール

### 育成対象と先頭ポケモンの決定（最重要）

**毎回のバトル後に**、以下の手順で育成対象と先頭ポケモンを判断し、宣言する。takanamito に聞かれる前に自分から宣言すること。

#### ステップ1: 育成対象を決める

以下の優先順位で育成対象を判断する。**単なるレベル差ではなく、この先の攻略への貢献度で決める。**

1. **次のジムリーダーや直近のボス戦で活躍できるポケモンを最優先**
   - タイプ相性で有利なポケモン、または弱点を突ける技を持つポケモン
   - 例: 次がでんきジム → じめん技を持つマンキーを優先育成。ピカチュウはでんきジムに不要なので後回し
2. **今後のストーリーで長期的に戦力になるポケモン**
   - パーティの中核として使い続ける予定のポケモンを優先
   - 一時的な戦力やニッチな役割のポケモンは後回し
3. **進化が近いポケモン**
   - あと数レベルで進化するなら優先度を上げる
4. **上記を満たす中で、レベルが低いポケモンを優先**
   - レベル差はあくまで同条件のポケモン間の判断材料

#### ステップ2: 次の相手を考慮して先頭を決める

1. **次のトレーナーの手持ちや、エリアの野生ポケモンのタイプを把握する**
2. **育成対象を先頭に出せるか判断する**
   - 育成対象が相手に弱点を突かれない → **育成対象を先頭にして、すぐ交代 or そのまま戦わせる**
   - 育成対象が相手に弱点を突かれる → **育成対象を先頭に出してすぐ交代**（1発食らうリスクがあるが経験値は入る）
   - 育成対象が相手に一撃で倒される危険がある（レベル差+弱点） → **別のポケモンを先頭にする**。育成対象は安全な相手のときに出す
3. **育成対象が先頭に出せない場合、次に育成が必要なポケモンや、相手に有利なポケモンを先頭にする**
4. **ライバル戦・ボス戦の直前 → エースを先頭にする**

#### ステップ3: 技の指示も一緒に宣言する

先頭ポケモンを宣言するとき、最初に使う技（または交代先）も一緒に伝える。

### バトル後の先頭宣言フォーマット

バトルが終わるたびに、次の形式で先頭を宣言する。**省略せず毎回宣言すること。**

```
【次のバトル準備】
回復: ○○にキズぐすり / 回復不要 / ポケモンセンターに戻る
次の先頭: ○○（Lv○○）
理由: ○○（レベルが一番低い / 進化まであとLv○○ 等）
相手の想定: ○○タイプが多い / 次のトレーナーは○○を使う
作戦: ○○先頭→○○に交代 / そのまま○○で攻撃
```

#### 回復判断の基準
- 先頭ポケモンのHPが**半分以下** → キズぐすり推奨
- 状態異常あり → 対応する回復アイテムを使う
- エースのHPが**3分の1以下** → 必ず回復してから進む
- 次がライバル戦・ボス戦 → 全員全快にする
- 回復アイテムの在庫が少ない場合 → ポケモンセンターや船内の回復NPCを利用

### エリア移動時

新しいエリアに入る前に、以下の手順で先頭ポケモンを決定する。

1. **エリアのトレーナー情報を把握する**（出現ポケモン・レベル帯・ライバル戦やイベント戦の有無）
2. **育成が必要なポケモンを特定する**（上記「育成対象の決定」に従う）
3. **育成ポケモンを先頭にできるか判断する**
   - ライバル戦やイベント戦がある/ありうるエリア → **エースを先頭にする**（育成より安全優先）
   - ただし、ライバル戦の直前まで育成ポケモンを先頭にし、ライバル戦前に入れ替えるのがベスト
   - ライバル戦がないエリア → **育成したいポケモンを先頭にする**
4. 育成ポケモンを先頭にする場合:
   - 最初のトレーナーの手持ちとの相性を確認
   - 1発は耐えられるレベル差か確認（レベル差5以上は危険）
   - 交代先として誰を控えるか事前に決めておく

### 経験値分配の効率的な方法

**育成したいポケモンを先頭 → すぐエースに交代**が最も効率的。
- 育成ポケモン: ダメージを受けずに経験値がもらえる
- エースポケモン: 交代で出て敵を倒し、経験値がもらえる
- 「エースで削ってから育成ポケモンに交代」は育成ポケモンがダメージを受けるリスクがある

### トレーナー戦の途中

- 次のポケモンが出てきたら、タイプを確認してから技を指示する
- 有利なポケモンが控えにいるなら交代を検討する
- HPが少ないポケモンは無理させず交代する

### 回答フォーマット（エリア移動時）

```
先頭: ○○
育成対象: ○○（理由: レベル差が最も大きい / 進化が近い 等）
作戦: ○○先頭→○○に交代 / そのまま戦わせる
控え: ○○に交代する想定
注意: ○○（ライバル戦の可能性がある場合等）
```
