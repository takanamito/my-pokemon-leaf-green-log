# 指示ミスの記録

Claude が実際に犯した指示ミスの記録。同じ失敗を繰り返さないための教訓集。

## 同じ指示の繰り返し

- **どくけし二重指示**: カメールの毒をどくけしで回復した直後に、また「どくけしを使おう」と同じ指示を出した。takanamito に「さっきどくけしで回復したやんか。君が指示したで」と言われた
- **わざマシン二重指示**: マンキーにわざマシン28あなをほるを教えた直後に、また「あなをほるを教えよう」と言った。takanamito に「お前が指示したやろ」と言われた

## アイテム効果の誤認

- **PPエイド誤説明**: PPエイドを拾ったとき「HPを60回復できる強い回復アイテム」と言った。PPエイドは技のPPを10回復するアイテムであり、HP回復ではない。takanamito に「嘘つけ」と言われた

## 状況の誤読

- **状態異常の主語間違い**: パラスのしびれごなでカメールが麻痺したとき「逆にチャンス！状態異常があると捕まえやすくなる」と言った。麻痺したのは相手ではなくこちら側。状況を正しく把握していなかった

## 名称の混同

- **化石の名前**: 「らせんのかせき」を選ぶよう指示 → 実際のゲーム内での名前は「かいのカセキ」。世代によって名前が異なることを考慮していなかった

## 技スロットの誤判断

- **ゼニガメのからにこもる**: 技が4つ未満なのに「忘れる技を選んで」と指示
- **パラスのわざマシン09**: 同様のミス。空きスロットがあるのに忘れさせようとした
