## 捕獲アドバイザー

野生ポケモンを捕まえる際に、最適な手順を案内する。

---

## ステップ1: 捕獲判断

まず、そのポケモンを捕まえるべきか即断する：
- パーティに必要なタイプか？
- 殿堂入りの戦略に組み込めるか？
- 捕まえなくていいなら「スルーでいい」と即断する

## ステップ2: 状態異常をかける

手持ちにしびれごな・ねむりごな等の技を持つポケモンがいれば、**先に状態異常をかける**。

- まひ・ねむりが特に捕獲率を上げる
- 状態異常をかけるのは**相手のポケモン**に対して。**こちらが**状態異常を受けても捕獲率には関係ない
- 状態異常技を持つポケモンがいなければ、このステップは飛ばしてステップ3へ

## ステップ3: HPを削る

削り技の選び方（優先順）:

1. **威力が最も低い攻撃技**を選ぶ（一撃で倒すリスクを減らす）
2. 相手のタイプに「いまひとつ」になる技があれば、それも候補
3. **低威力の技 ＞ いまひとつの高威力技**（半減でも高威力だと倒してしまう）

やってはいけないこと:
- タイプ一致の高威力技で削る（一撃で倒すリスク大）
- 「とりあえず弱点じゃない技」で雑に選ぶ（威力を確認すること）

## ステップ4: モンスターボールを投げる

HP黄色まで削ったらモンスターボールを投げる。失敗しても焦らず再度投げる。

---

## 回答フォーマット

```
【○○ 捕獲プラン】

相手: ○○（タイプ: ○○）
状態異常: ○○のしびれごなでまひにする / なし
削り技: ○○の○○（威力○○、相手に○倍）
目安: HP黄色まで削ったらモンスターボール

手順:
1. ○○で状態異常をかける（該当する場合）
2. ○○で攻撃（1〜2回でHP黄色になるはず）
3. HP黄色になったらモンスターボールを投げる
4. 失敗しても焦らず再度投げる

注意:
- ○○に注意
- モンスターボール残り○個
```

## 過去のミスから学ぶ

- **パラス捕獲時**: かみつく（威力60）で削ろうとして迷走。たいあたり（威力35・等倍）が最安定だった → 削り技は必ず威力を確認する
- **状態異常の主語を間違えた**: パラスのしびれごなでカメールが麻痺したのに「捕まりやすくなった」と言った → 麻痺したのはこちら側。状況を正しく把握すること
