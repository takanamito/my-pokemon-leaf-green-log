## Claude 頻出ミス防止チェッカー

Claude が過去に何度も繰り返してきたミスのパターンと防止策。
**ゲームに関するアドバイスをする前に、該当するチェック項目を必ず確認すること。**

---

## ミスカテゴリ一覧

### 1. ポケモンのタイプを間違える

**過去の事例:**
- サンドを「じめん/いわ」と言った → 実際は**じめん単タイプ**
- アーボを「ノーマル/どく」と言った → 実際は**どく単タイプ**
- コイル戦で「でんきはみずに等倍」と言った → 実際は**ばつぐん**

**防止策:**
- ポケモンのタイプに言及する前に、`type.md` の相性表で確認する
- **確信がない場合は断言しない。「確認するね」と一言添える**
- 特に複合タイプか単タイプかを間違えやすい。よくある誤認パターン:
  - サンド → じめん（いわではない）
  - ズバット → どく/ひこう
  - コイル → でんき/はがね
  - ヤドン → みず/エスパー

---

### 2. アイテムの効果を間違える

**過去の事例:**
- ピーピーエイドを「HPを60回復できる強い回復アイテム」と言った → 実際は**技のPPを10回復するアイテム**
- 化石の名前を間違えた（「らせんのかせき」→ 正しくは「かいのカセキ」）

**防止策:**
- アイテムの効果を説明するとき、**確信がなければ「正確な効果はちょっと自信ないから確認して」と正直に言う**
- 特に間違えやすいアイテム:
  - ピーピーエイド: PP10回復（HPではない）
  - ピーピーリカバー: PP全回復
  - ピーピーマックス: 全技PP全回復
  - キズぐすり: HP20回復
  - いいキズぐすり: HP50回復
  - すごいキズぐすり: HP200回復
  - げんきのかけら: 瀕死から復活しHP半分回復
  - げんきのかたまり: 瀕死から復活しHP全回復
  - ふしぎなアメ: レベルを1上げる
- **「〜だと思う」ではなく、知っていることだけを断言する**

---

### 3. 技スロットの空きを確認しない

**過去の事例（2回繰り返し）:**
- ゼニガメが技2つ（たいあたり/しっぽをふる）のときに「からにこもるを覚えるなら、しっぽをふるを忘れよう」→ **技枠4つあるから忘れる必要なし**
- パラスが技2つ（ひっかく/しびれごな）のときに「タネマシンガンを覚えるなら、ひっかくを忘れさせよう」→ **技枠4つあるから忘れる必要なし**

**防止策: 技の習得・忘れに関する発言をする前に、以下を必ず確認する:**

```
□ そのポケモンは今、技を何個覚えているか？
□ 4個未満なら → 忘れる必要なし。「そのまま覚えよう」と言う
□ 4個なら → 入れ替え候補を実効威力で比較して提案する
```

**4個未満なのに「忘れよう」と言うのは絶対禁止。**

---

### 4. 自分が出した指示を忘れる

**過去の事例:**
- カメールのどくを「どくけしで治そう」と指示 → 治した直後に「どくけし使おう」と再度指示。takanamito に「さっき指示したやんか」と言われる
- マンキーにあなをほる（わざマシン28）を覚えさせた → 直後に「マンキーにあなをほるで！」と指示しようとして知らなかったかのように振る舞い、「お前が指示したやろ」と突っ込まれる

**防止策:**
- 自分が直前にした指示を忘れていないか、発言前に会話を振り返る
- 特に「アイテムを使った」「技を覚えさせた」などの変更を伴う指示は、その後の会話に反映させる
- 不安なら「さっき○○したよね？」と確認してから話す

---

### 5. スキルの使い忘れ

**過去の事例:**
- オツキミヤマ〜4番道路で `/prep` を使い忘れた。「なんでさっき言ったときにskill使わなかったの？」と指摘された

**防止策: 以下のタイミングでスキルを使うこと:**

| タイミング | 使うスキル |
|-----------|-----------|
| 新しいエリアに入る前 | `/prep`（準備チェック） |
| バトル中の技選択 | `battle.md` を参照 |
| タイプ相性の質問 | `type.md` を参照 |
| ポケモンを捕まえるとき | `/catch`（捕獲アドバイス） |

**特に `/prep` は「次のエリアに向かう」という話になった時点で即座に使う。後回しにしない。**

---

### 6. バトル後のアイテム補充を言わない

**過去の事例:**
- ライバルシゲキ戦でいいキズぐすりをラスト1個まで使い切ったのに、補充を言わずに出発しようとした。takanamito に「アイテムは買っていかなくていいの？」と指摘された
- ハナダシティで普通のキズぐすりを買い忘れた

**防止策:**
- 消耗の激しいバトル（ジム戦・ライバル戦・長期ダンジョン）の後は、**必ずアイテム残数を確認する**
- 以下のアイテムが減っていたら補充を提案する:
  - キズぐすり / いいキズぐすり: **3個以下なら補充**
  - どくけし / まひなおし: **1個以下なら補充**
  - モンスターボール: 捕獲予定があるなら **5個以上キープ**

---

## 根本原則

1. **知ったかぶりをしない** — 確信がないことは「確認するね」と言う。間違った断言が一番信頼を失う
2. **技スロットは4つ** — 忘れさせる前に必ず今の技数を確認する
3. **スキルは後回しにしない** — 使うべきタイミングが来たら即座に使う
4. **自分の指示を覚えておく** — 直前の会話を振り返ってから発言する
5. **バトル後は必ず棚卸し** — アイテム残数・パーティのHP・状態異常を確認する
