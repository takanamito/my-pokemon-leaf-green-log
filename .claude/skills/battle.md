## バトル技選択・先頭ポケモン決定スキル

バトル中の技選択と、エリア移動時の先頭ポケモン決定を行う。

## 過去のミス（繰り返さないこと）

- ライバルシゲキ戦: マンキーLv10を先頭にしたままライバル戦に突入。Lv17ピジョンに何もできず交代する羽目に
- フシギダネ戦: パラスを出してねむりごなを先制で撃たれ、やどりぎ＋ねむりコンボにハマった
- ポッポ戦: からてチョップ（かくとう）をひこうタイプに推奨。かくとうはひこうに半減
- ナゾノクサ戦: カメールのみずのはどう（みず→くさは半減）を推奨。かみつく（等倍）の方が上
- コラッタ戦: ひっかく（威力40）を推奨。タネマシンガン（威力25×2〜5、最低50）の方が上
- 等倍の相手に対して、STAB付きのみずのはどう（実効90）ではなくかみつく（実効60）を何度も推奨

## 毎回間違えるタイプ相性NG例（技を指示する前に必ず確認）

### くさ技（タネマシンガンなど）
- **いわタイプにばつぐん**（×2）← 「いわに半減」は絶対に間違い
- じめんタイプにもばつぐん（×2）
- みずタイプにもばつぐん（×2）
- ひこう・むし・どく・ほのお・くさ・ドラゴン・はがねにいまひとつ
- **バタフリー（むし/ひこう）にタネマシンガン推奨は最悪（0.25倍）**

### かくとう技（からてチョップ・けたぐりなど）
- **ひこうタイプにいまひとつ**（×0.5）← ひこうに推奨しない
- むし・どく・エスパーにもいまひとつ
- いわ・ノーマル・こおり・あく・はがねにばつぐん

### みず技（みずのはどうなど）
- **くさタイプにいまひとつ**（×0.5）← くさ相手にみず技は損
- みずタイプにもいまひとつ
- くさ相手には等倍のかみつく（実効60）の方がみずのはどう（実効45）より上

### よく間違えるポケモンのタイプ
- **サンド/サンドパン: じめん単タイプ**（いわなし）← かくとうは等倍
- **ナゾノクサ: くさ/どく**← みず半減、かみつくが正解
- **ヤドン/ヤドラン: みず/エスパー**← かみつく（あく→エスパーばつぐん）が正解

### からてチョップとけたぐり
- **両方ともかくとうタイプ**。タイプが違うわけではなく威力の差のみ

## 技選択ルール

### 実効威力の計算（最重要）

技を推奨する前に、**必ず実効威力を計算する**。

```
実効威力 = 技の威力 × STAB × タイプ相性

STAB: 技のタイプ = 使うポケモンのタイプ → ×1.5、それ以外 → ×1.0
タイプ相性: ばつぐん → ×2.0、等倍 → ×1.0、いまひとつ → ×0.5、こうかなし → ×0
複合タイプ: 各タイプの倍率を掛け合わせる（例: いわ/じめん にみず → 2.0 × 2.0 = 4.0）
```

### 計算例（現パーティ）

**カメール（みずタイプ）が等倍の相手を攻撃する場合:**
- みずのはどう: 威力60 × STAB1.5 × 等倍1.0 = **実効90** ← これが最強
- かみつく: 威力60 × 1.0 × 1.0 = **実効60**
- たいあたり: 威力35 × 1.0 × 1.0 = **実効35**

**パラス（むし/くさタイプ）が等倍の相手を攻撃する場合:**
- タネマシンガン: 威力25×2〜5回 × STAB1.5 = **最低実効75**（2回命中）〜**最大実効187.5**（5回命中）
- ひっかく: 威力40 × 1.0 = **実効40**

**マンキー（かくとうタイプ）がいわタイプを攻撃する場合:**
- からてチョップ: 威力50 × STAB1.5 × ばつぐん2.0 = **実効150**
- ひっかく: 威力40 × 1.0 × 1.0 = **実効40**

### 技選択の手順

1. 相手のポケモンのタイプを確認する（複合タイプも忘れずに）
2. 使えるすべての攻撃技の**実効威力を計算**する
3. **最も実効威力の高い技**を選ぶ
4. 実効威力が同じ場合は、命中率が高い方を選ぶ
5. PPが残り少ない場合は、2番目の候補を検討する

### 状態異常技の活用判断

- 相手のレベルが高い/HP管理が厳しい → しびれごな等でまひにして安全に
- 相手のレベルが低い/余裕がある → 状態異常技は不要、攻撃で倒す
- 捕獲時はしびれごな・ねむりごなが最優先

## 先頭ポケモン決定ルール

### エリア移動時

新しいエリアに入る前に、以下の手順で先頭ポケモンを決定する。

1. **エリアのトレーナー情報をWebで調べる**（出現ポケモン・レベル帯・ライバル戦やイベント戦の有無）
2. **育成が必要なポケモンを特定する**
   - パーティ内でレベルが低いポケモンはいるか？
   - 進化が近いポケモンはいるか？
   - トレーナー戦の経験値で効率よく育てられるか？
3. **育成ポケモンを先頭にできるか判断する**
   - ライバル戦やイベント戦がある/ありうるエリア → **エースを先頭にする**（育成より安全優先）
   - ライバル戦がないエリア → **育成したいポケモンを先頭にする**
4. 育成ポケモンを先頭にする場合:
   - 最初のトレーナーの手持ちとの相性を確認
   - 1発は耐えられるレベル差か確認（レベル差5以上は危険）
   - 交代先として誰を控えるか事前に決めておく
   - 戦闘開始→すぐ交代の流れで経験値を分配する（先頭に出ただけで経験値がもらえる）

### トレーナー戦の途中

- 次のポケモンが出てきたら、タイプを確認してから技を指示する
- 有利なポケモンが控えにいるなら交代を検討する
- HPが少ないポケモンは無理させず交代する

### 回答フォーマット（エリア移動時）

```
先頭: ○○
理由: ○○（次のトレーナーの手持ちが○○タイプだから / 育成したいから等）
控え: ○○に交代する想定
注意: ○○（ライバル戦の可能性がある場合等）
```
