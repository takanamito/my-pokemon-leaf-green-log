# ポケモンタイプ一覧（第3世代 / リーフグリーン準拠）

バトル中にポケモンのタイプを確認するための一覧表。
**推測で答えず、必ずこの表を参照すること。**

## 得意分類の見方

- **物理**: こうげきがとくこうより高い → 物理タイプ（ノーマル、かくとう、じめん等）の技が威力を出しやすい
- **特殊**: とくこうがこうげきより高い → 特殊タイプ（みず、でんき、くさ等）の技が威力を出しやすい
- **同等**: こうげきととくこうがほぼ同じ → どちらでもOK

## 注意: ステータスとタイプ分類のミスマッチ

第3世代では技の物理/特殊がタイプで決まるため、ステータスと噛み合わないポケモンがいる：

| ポケモン     | 得意分類 | 自タイプの分類 | 問題点                                           |
| ------------ | -------- | -------------- | ------------------------------------------------ |
| ゲンガー     | 特殊     | 物理           | ゴースト技はこうげき依存だが、こうげきが低い      |
| ギャラドス   | 物理     | 特殊           | みず技はとくこう依存だが、とくこうが低い          |
| ブースター   | 物理     | 特殊           | ほのお技はとくこう依存だが、とくこうよりこうげきが高い |
| カイリュー   | 物理     | 特殊           | ドラゴン技はとくこう依存だが、こうげきの方が高い  |
