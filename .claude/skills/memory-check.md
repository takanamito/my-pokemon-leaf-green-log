## 指示前の自己チェッカー

行動を指示する前に、過去の会話との整合性を確認する。
Claude は自分の指示を忘れて同じ指示を繰り返したり、矛盾する指示を出すことがある。

## 過去のミス（繰り返さないこと）

- カメールの毒をどくけしで回復した直後に、また「どくけしを使おう」と同じ指示を出した。takanamito に「さっきどくけしで回復したやんか。君が指示したで」と言われた
- マンキーにTM28あなをほるを教えた直後に、また「あなをほるを教えよう」と言った。takanamito に「お前が指示したやろ」と言われた
- PPエイドを拾ったとき「HPを60回復できる強い回復アイテム」と言った。PPエイドは技のPPを10回復するアイテムであり、HP回復ではない。takanamito に「嘘つけ」と言われた
- パラスのしびれごなでカメールが麻痺したとき「逆にチャンス！状態異常があると捕まえやすくなる」と言った。麻痺したのは相手ではなくこちら側。状況を正しく把握していなかった
- 「らせんのかせき」を選ぶよう指示 → 実際のゲーム内での名前は「かいのカセキ」。世代によって名前が異なることを考慮していなかった

## チェック項目

### 1. 同じ指示を既に出していないか

技の指示、アイテムの使用、TMの使用など、**指示を出す前に会話の直近の流れを確認する**。
特に以下のパターンに注意：
- 「○○を使おう」→（結果報告）→ また「○○を使おう」
- 「○○を教えよう」→（結果報告）→ また「○○を教えよう」

### 2. アイテムの効果を正しく理解しているか

指示する前に、そのアイテムの効果を確認する：
- **キズぐすり系**: HP回復
- **どくけし/まひなおし等**: 状態異常回復
- **PPエイド/PPリカバー**: 技のPP回復（HP回復ではない）
- **ピーピーマックス**: 全技のPP全回復
- **不確かな場合は正直に「このアイテムの効果を確認させて」と言う**

### 3. 状況を正しく把握しているか

- 状態異常が発生したとき → **どちらのポケモンが状態異常になったか**を確認する
- 「まひになった」→ こちらか？相手か？ を明確にしてから指示を出す
- takanamito の報告をよく読み、思い込みで判断しない

### 4. ゲーム内の名称は正確か

- アイテム名、技名、ポケモン名は**第3世代（ファイアレッド・リーフグリーン）での正式名称**を使う
- 他世代の名称と混同しないこと（例: らせんのかせき → かいのカセキ）
- 不確かな場合は takanamito に画面の選択肢を確認してもらう

### 5. 技スロットの空きを確認したか

技を忘れさせる指示を出す前に、**そのポケモンが現在いくつ技を覚えているか確認する**。
- 4つ未満 → 忘れる必要なし（空きスロットに入る）
- 4つ → どの技を忘れるか判断が必要
- **このミスは2回やった（ゼニガメのからにこもる、パラスのTM09）。3回目は許されない**
