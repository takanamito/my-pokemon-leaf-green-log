# 029 - 四天王リベンジと殿堂入り — シゲキ撃破で頂点へ

## 日時

2026-04-21

## 現在地

セキエイこうげん（殿堂入り達成）

## 攻略ログ

### 再挑戦前の仕込み — パーティ刷新とレベル上げ

028 のシバ全滅から立て直し。前回の反省点は「ナッシー Lv40・カビゴン Lv43・ゲンガー Lv46 のレベル不足」と「回復アイテム不足」の2点だった。

ファイヤーをボックスに戻し、代わりに**ドードリオ Lv49**をパーティに加えてパーティを刷新。チャンピオンロード周辺でレベル上げを行い、ナッシー Lv50・ゲンガー Lv52・カメックス Lv54・ニドキング Lv54 まで引き上げた。回復アイテムも補充して、満を持して四天王へ再挑戦。

### カンナ戦（こおり使い）— ちょすいラプラスと猛毒メロメロ

Claude の作戦: みず/こおり勢にはゲンガーの10まんボルトが×2、ヤドランにはシャドーボールが×2。ラプラスはカメックスのなみのりで受けるプランだった。

ところが takanamito から一言：「**ラプラスはちょすい**」。みずタイプを無効化する特性。カメックスのなみのり封じという根幹を覆す一言で、Claude の作戦が崩壊した。ラプラスにはなみのりが通らないどころか、むしろ回復させてしまう。

急遽作戦を修正。みず技全面禁止、ラプラスには**ゲンガーの10まんボルト×2**で削るプランに切り替えた。

- ジュゴン → ゲンガーの10まんボルトで一撃
- パルシェン → ゲンガーの10まんボルトで処理
- ヤドラン → ゲンガーのシャドーボール×2 STAB で撃破
- ルージュラ → 戦闘中に**メロメロ**を食らった。カメックス♂がルージュラ♀のメロメロ状態に。50%で動けないまま削り合いは危険と判断し、ゲンガーに交代してメロメロを解除。シャドーボールで仕留めた
- ラプラス → ゲンガーの10まんボルト×2でHP管理しながら削り切り

前回は2匹落とした難所を、今回は全員生存で突破。パーティレベルの差と特性の把握が明暗を分けた。

### シバ戦（かくとう使い）— 完封リベンジ

前回はカメックス単騎で挑んで全滅した相手。今回は作戦が完璧だった。

- イワーク2匹 → **カメックスのなみのり×4**で即ワンパン。水に岩と地面の4倍弱点は反則
- かくとう勢3匹 → **ゲンガー**に交代。ゴーストタイプにかくとう技は無効なので、一方的にシャドーボールで削る

前回全滅の原因だったエビワラーも今回はゲンガーの前に為す術なし。エビワラーにふぶきぶち込んで「**寒そう**」と言っていた前回とは大違い。カメックスとゲンガーの2枚だけで完封した。

### キクコ戦（ゴースト使い）— ふゆうに気づかなかった司令塔

Claude の当初作戦：「ゲンガー（ふゆう持ち）にはじしん無効だが、キクコのゲンガーはじめん2倍弱点なのでニドキングのじしんSTABでワンパン」。

takanamito の一言：「**ゲンガーはふゆう持ち**」。

……キクコのゲンガーにも同じ特性が適用される。じしんは完全無効。相手のゲンガーに使わせようとした技が効かない。シリーズでも屈指の凡ミス。前回の「わすれおやじ2の島事件」に続く基本知識の欠落だった。

作戦を全面修正。ゲンガー（ゴースト/どく）全員に**エスパー×2**で刺さる**ナッシーのサイコキネシスSTAB**が主力になった。

- ゲンガー Lv56 → ナッシーのサイコキネシスで一撃
- ゴルバット → ゲンガーの10まんボルト×2（ひこう弱点、じしん無効）
- ゴースト → ナッシーのサイコキネシスで処理
- アーボック → ニドキングのじしん×2でワンパン
- エースゲンガー Lv60 → ナッシーのサイコキネシスで撃破

「**ナッシーのサイコキネシスでいく**」という takanamito の一言で作戦決定。ジム開始時のじしん作戦崩壊にも動じず、ナッシーで押し切った。

### ワタル戦（ドラゴン使い）— ふぶき解禁とカイリュー一撃

カンナ戦から3戦温存してきたカメックスの**ふぶき**を、ここで全力解放。

- ギャラドス → ゲンガーの10まんボルト×4（みず/ひこうは4倍弱点）でワンパン
- ハクリュー1・2 → カメックスのふぶき×2でそれぞれ仕留める
- プテラ → カメックスのなみのり×2 STAB（いわ弱点）でワンパン。ふぶきをカイリュー用に温存
- カイリュー → **カメックスのふぶき急所一撃**

ドラゴン/ひこうで4倍弱点のカイリューへ、ふぶきが急所に直撃。あの温存計画が報われた瞬間だった。

### チャンピオン・シゲキ戦 — パーティ予想半分外しと殿堂入り

ワタルを退けてチャンピオンの間へ。扉の前に立っていたのは、旅の始まりからずっと行く先々で現れていたライバルのシゲキだった。takanamito の「**さっき戦ったのにね**」という一言が全てを言い表していた。

Claude はカメックスを選択したプレイヤーの対戦相手として以下のパーティを予想していた：ピジョット・フーディン・**ギャロップ・スターミー・パルシェン**・フシギバナ。

実際のパーティ：ピジョット・フーディン・**サイドン・フシギバナ・ギャラドス・ウインディ**。

**6体中3体を外した**。ギャロップ・スターミー・パルシェンの3体がまるごと違う。シゲキのパーティを半分も把握できていない司令塔。

- ピジョット → **ゲンガー**の10まんボルト×2で撃破
- フーディン → **ニドキング**のメガホーン×2（むし→エスパー）でワンパン
- サイドン（想定外登場）→ takanamito が「**サイドンきた**」と報告。**カメックスのなみのり×4**でワンパン。みず/じめん4倍弱点かつ素早さで先制
- フシギバナ（エース予想だったが4番手）→ **ナッシーのサイコキネシス**を2発で撃破。どく弱点×2 STAB が炸裂
- ギャラドス（想定外）→ **ゲンガーの10まんボルト×4**で削り始めるが、**かいふくのくすり3連発**で猛烈に粘られる。削っても削っても回復される泥沼戦に。takanamito の「**かいふくのくすり3連発でクソ粘られてる**」の一言がそのまま状況を伝えていた。粘りに耐えられなくなったところで**ゲンガーのみちづれ**を発動。ギャラドスを道連れに相打ち
- ウインディ（想定外）→ takanamito が「**ウインディ来ると思うからカメックスにしとく**」と自ら予測。Claude はここだけ予想を外していない。**カメックスのなみのり×2 STAB**でフィニッシュ

**ウインディを倒した瞬間、殿堂入り達成。**

takanamito のパーティ6匹が、オーキド博士の前で壁に映し出された。

### 殿堂入りとエンドロール

オーキド博士の登場、殿堂入りマシンの起動、6体のポケモンが壁にフラッシュバックで映し出されるシーン。

エンドロールについて、takanamito の感想は「**エンドロールおしゃれ**」。主人公とポケモンのシルエットが横スクロールで流れ、音楽がマサラタウンBGMのアレンジで「旅の始まりに帰る」構成。

### 殿堂入り後 — カメックスのロケットずつき議論

殿堂入り直後、takanamito が「**カメックスにロケットずつきを覚えさせたい**」と提案。

Claude の評価：ロケットずつきは溜め技（1ターン目準備→2ターン目攻撃）でタイプ不一致のノーマル物理。カメックスは特殊アタッカー寄りで実用性は低い。忘れさせるなら最初は**ハイドロカノン**が候補だったが、takanamito が「**ハイドロカノンの方を優先します**」と方針転換。ハイドロカノンを残す場合は**かみつく**を忘れる方向で検討したものの、最終的に「**ロケットずつき諦めました**」で決着。カメックスの技構成は変わらずで今回の冒険を締めくくった。
