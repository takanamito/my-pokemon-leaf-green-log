# 026 - チャンピオンロード突破

## 日時

2026-04-11

## 現在地

セキエイこうげん入口（ポケモンリーグ直前）

## 攻略ログ

### トキワシティ発 — ライバル戦の甘い見立て

- トキワジムを攻略した後、まずはトキワシティで準備。回復は済ませたものの、まんたんのくすりやげんきのかたまりといった四天王戦用アイテムの買い出しは後回しにして、takanamito は「先にチャンピオンロード行く」と前のめり
- Claude はレベル上げ方針として、一番低いナッシー（Lv37）を先頭にしてXP稼ぎを提案。じんすけはレベル上げ対象外、がくしゅうそうちはゲンガーへ
- 22番道路に入るや否や、**ライバル・シゲキ**が立ちはだかる。初手ピジョット。ナッシー（くさ/エスパー）にとんでもなく不利な相性で、takanamito が「**だからさっき先頭聞いたのにさ**」と苦言。ライバル戦の気配を読めずナッシー先頭にした Claude の完敗
- 慌ててニドキングに交代 → 10まんボルトでピジョットを一撃。なんとか立て直す

### シゲキ戦 — かなしばりとみちづれ習得の劇的展開

- 2匹目はフーディン。Claude は「カメックスに交代してかみつく（あく技）」を指示。ニドキングはエスパー2倍弱点、フーディンは素早さが高いので危険と判断
- takanamito が「**このままメガホーンは？**」と食い下がる。メガホーン自体は抜群だが先制されて耐えられない可能性が高いので、Claude は安全策のカメックス交代を推奨
- カメックス → かみつく、と思いきや**かなしばりでかみつく封印**。厄介な展開。なみのりに切り替えてごり押し
- フーディンを下げてフシギバナ登場。カメックスは草4倍弱点なので**カビゴンに交代**。takanamito が慎重に凌ごうとしたが、**カビゴン瀕死**。フシギバナはHP1mmでこちらに粘られ、**ニドキング → じしん**でギリギリ仕留めた
- 再びフーディンが出てきたので、一度下げたカメックスでかなしばり解除済みの**かみつく**を決めて撃破。続くガーディ、サイホーンもカメックスのなみのり一撃。ガーディの「ほのお→4倍」、サイホーンの「じめん+いわ4倍弱点」で気持ちよく一掃

![ニドキング vs ポニータ](../images/026_champion_road/026_02_nidoking_earthquake_vs_ponyta.jpeg)

- ゲンガーがLv39で**みちづれ**を習得。四天王の詰み回避用に取っておきたい切り札として、ナイトヘッドと置き換えて採用
- ラストのギャラドスは本命の**ゲンガー → 10まんボルト**。takanamito が「**ハイドロカノンや！！！！**」とカメックスに切り替えて圧倒的一撃で葬る。個人的にも気持ちよかったらしく、報告のテンションが一段上がっていた
- 道中で野生のオニドリルを捕獲成功。シゲキ戦で消耗したがギリギリで22番道路を抜けた

### チャンピオンロード突入 — prepスキルの反省

- takanamito が「チャンピオンロード突入」と宣言。Claude は急いで prep スキルを実行して「まんたんのくすり、どくけし、あなぬけのヒモを買いに戻るべき」と進言
- takanamito 「**四天王じゃなくてチャンピオンロードなんやけど**」。Claude が四天王戦と勘違いして過剰な準備を要求していた。「ポケモンリーグ直前の岩山みたいなダンジョン」という本人の説明でようやく認識が合った。結局ハイパーボール×8とすごいキズぐすり×2を買って突入

![チャンピオンロード入口](../images/026_champion_road/026_01_champion_road_entrance.jpeg)

### 炎トレーナー無双

- いきなり**エリートトレーナー**。しかも視界外から補足してきて takanamito が「**こいつら視界の外から補足してくる**」と苦笑
- ペルシアンをニドキングで処理。続くポニータ、ロコン、ギャロップ、キュウコン、そしてラスト。全ポケモンがほのお系で、takanamito が「**こいつほのおだらけ**」とツッコむ始末。カメックスのなみのりと、ニドキングのじしんで完封
- ポニータにはじしん、ギャロップ以降はカメックスで、と takanamito が「**まんべんなくレベル上げたいから**」と指示を出して役割分担

### フシギソウ → リザードン — 読みが冴えた瞬間

- エリートトレーナー再戦。フシギソウにはゲンガーのシャドーパンチで対応し、XPもゲンガーに入れる
- 続けて**リザードン登場**。takanamito 「**リザードン来ちゃった…**」。ゲンガーのまま10まんボルトで2倍抜群を狙う Claude に対し、takanamito 「**カメックスだろ**」とより確実な選択肢を指摘
- ハイドロカノンが気持ちよく決まり、リザードンを粉砕。「ハイドロカノン」の威力と爽快感が Claude との協力でうまくハマった場面

### 名言「トレーナーの最後の試練」

- 道中、**カラテおうのタダシ**が「**チャンピオンロードはトレーナーの最後の試練らしいよ**」と教えてくれる。ここを抜ければポケモンリーグという、エリア全体の締めにふさわしいフレーバーで、takanamito もこの台詞が気に入った様子

### 水・電気・草 — ニドキング無双

- ゴルダックにナッシーのギガドレインで水2倍抜群＆HP回復
- 野生サンドパンを捕獲しようとしたが takanamito「**あー倒しちゃった**」。ありがち
- ラプラス戦。takanamito が「**みず技怖いからゲンガー**」と自主判断。ゴースト/どくは水/氷両方とも等倍で受けられるのでナイスコール。10まんボルトで電気2倍抜群
- ベロリンガにはニドキングのかわらわり（格闘2倍）で対応。ゲンガーのシャドーパンチはノーマルに無効なので交代必須
- 野生のレベル高いイワークが頻繁に出てくることに takanamito が「**この辺やたらレベル高いイワーク出てきてうざい**」と愚痴。たしかに面倒

### ゲンガー Lv45 — シャドーボール習得

- ゲンガーが**シャドーボール**を習得。Gen3仕様でゴースト技はすべて特殊扱い。威力60のシャドーパンチに対して威力80、さらに特防ダウンの追加効果付きで上位互換
- シャドーパンチを上書きしてシャドーボールに差し替え。ゲンガーの主力がまた一段強くなった

### 電気無効を忘れる Claude

- エリートトレーナーのウツボット系草タイプ連戦。Claude がニドキングのメガホーン2倍で押していたが、**メガホーンのPP切れ**が判明
- 次善策として Claude が「**10まんボルト**」を指示 → takanamito 「**あいて地面やんか**」。相手のマダツボミ/ウツドン/ウツボットはくさ/どくなので地面じゃないが、そもそも Claude はニドキング自身の電撃が地面タイプ（相手）に無効になる場合と混同して口が滑った
- じしんやかわらわりで耐える展開。そのうちメガホーンを外した隙に**しびれごな**をくらい、**ねむりごな** → カビゴンでねごと連打の耐久戦に突入。カビゴンが眠ったまま「おんがえし」を引いて見事に勝利

### ダブルバトル — じしんで自爆寸前

- エリートカップルとのWバトル。相手はニドキング＋ニドクイン、こちらはニドキング＋カメックス
- Claude の指示は「**ニドキングは10まんボルト**で集中砲火、カメックスはふぶきで全体」だったが、takanamito 「**あいて地面やんか**」と即座にツッコミ。相手は地面タイプで電気無効。Claude が2度目の地面無効見落とし。メガホーンに変更するも**いまひとつ**。ここまで来て完全に手詰まり
- 仕方なく**ニドキングはじしん**を選択。じしんは全体攻撃なので**味方のカメックスにも当たる**。takanamito 「**じしんでカメックス死にかけた**」と悲鳴
- 運よく相手のニドキングが**ふぶきでこおり状態**になって動けず、助かった。なんとか勝利したが、ダブルバトルの全体攻撃周りで Claude がダメージ計算を詰めきれずヒヤヒヤさせた場面
