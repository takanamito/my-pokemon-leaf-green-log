# 007 - ハナダシティ南下〜6番道路

## 日時
2026-03-03

## 現在地
6番道路（タマムシシティ手前）

## 攻略ログ

- マサキの家を出発し、24番道路（ゴールデンブリッジ）を南へ戻る。前回戦い残していたトレーナーがいたので倒していく

**短パンこぞうのカツユキ**戦。アーボが出てきたのでマンキーのけたぐりを指示したが、Claude が「アーボはノーマル/どくタイプだからからてチョップより等倍」と余計な説明をつけた直後、takanamito に「**空手チョップとけたぐりは両方ともかくとう**」と指摘される。言われた通り。2つともかくとうタイプで威力の違いしかない。

続いてサンドが登場。Claude が「サンドはじめん/いわタイプだからかくとうがばつぐん」と自信満々に指示を出すと、「**等倍だったよ**」と報告が来た。サンドはじめんタイプのみで、かくとうは等倍。またやった。

- ハナダシティに戻り全員回復。南へ向かうと謎の木（きりかぶ）があって道をふさがれている。きりさくがなければ通れないため別ルートを探す

自転車屋で100万円の自転車を目撃。さすがに買えない。後で無料入手ルートがある。

- 5番道路を南下。ちかつうろを通って南へ抜けるルートを発見。地下道を通って6番道路へ

途中、そだてや（育て屋）を発見。今は預けず先へ進む。

- 6番道路の民家でロケット団に荒らされた家を発見。家の裏に大きな穴が開いており、降りてみるとロケット団員が潜伏していた

バトル開始。**ワンリキー**はカメールのみずのはどうで撃破、**スリープ**はカメールのかみつく（あくがエスパーにばつぐん）でさいみんじゅつを食らいながらも撃破。ロケット団員を倒すと**わざマシン28（あなをほる）**を入手！

- あなをほるをマンキーに覚えさせる。にらみつけると入れ替えてマンキーの技構成が強化された

マンキーの技構成：みだれひっかき / けたぐり / からてチョップ / あなをほる（※この時点ではみだれひっかきはまだ習得前）

- 6番道路でトレーナー育成戦を開始。マンキー先頭で進む

**むしとりしょうねんテツジ**戦。ビードルが出てきたので「からてチョップで攻めよう」と指示した直後に「**いまひとつ**」の報告。かくとうはむしタイプに半減だった。タイプ相性ミスが今日も出る。ひっかくに切り替えて対処。

続くビードル戦でマンキーのHPが6まで削られ、カメールのみずのはどうで撃破。

- マンキーが**Lv16**にレベルアップ！**みだれひっかき**を習得し、ひっかくと入れ替え。複数回ヒットする技になって火力が上がった

- **キャンプボーイのユウタ**戦。ゼニガメが出てきたのでパラスのタネマシンガン（くさがみずにばつぐん）で対応。「かてえ」とのことだったが、タネマシンガンを連打して撃破

- 女トレーナー戦。**コラッタ**はマンキーのからてチョップで一撃。続く**ピカチュウ**はマンキーのあなをほる（じめんがでんきにばつぐん・実効120）で一撃撃破！

しかしバトル後、マンキーがピカチュウの特性**せいでんき**でまひ状態に。まひなおしで回復した。

- **ノブユキのバタフリー**戦。これが今日最大の地獄だった

バタフリーはむし/ひこうタイプで、マンキーのかくとう技は両タイプに半減で実効18.75しか出ない。カメールのみずのはどう（実効90）で攻めようとしたが……

カメールが**超音波→しびれごな**のコンボを食らって混乱＋まひに。Claude が「パラスに交代してタネマシンガンで」と指示したが、タネマシンガン（くさ）はひこうタイプに半減でむしタイプにも半減、実効22.5しか出ない大チョンボ。takanamito はその通り実行してくれたが当然ダメージが入らず苦戦。

Claude が「やっぱりカメールに戻してまひなおし使って」と指示を変更。パラスが混乱した状態でカメールに交代し、まひなおしを使ってからみずのはどうで反撃。しかし今度はバタフリーが**どくのこな**でカメールを毒状態に。さらに**ねむりごな**でカメールが眠ってしまう。

ピカチュウに交代してでんきショックを試みたが**ねむりごな**でそのまま眠らされる。

マンキーに交代してみだれひっかき連打。混乱しながらも50%を祈って攻撃を続け、バタフリーのHPを残り1まで削ったところでマンキーが力尽きる。

最後はパラスを投入。Claude は「タネマシンガンは0.25倍だから絶対NG、ひっかく（ノーマル・等倍）一択」と今更正しい指示を出す。パラスが混乱しながらもひっかくでバタフリーを撃破！

「**よくやった！ただしパーティがボロボロ**」と自分でまとめるしかなかった。

バトル終了時のパーティ状態：
- マンキー：瀕死
- カメール：毒＋ねむり
- ピカチュウ：ねむり
- パラス：なんとか生存

どくけしでカメールの毒を応急処置し、ポケモンセンターへ急ぐ途中——

- ピカチュウが**でんじは**を習得！なきごえと入れ替えで覚えさせた。まひを入れる技は捕獲にもバトルにもかなり役立つ

- そして、ポケモンセンターまであと少しのところで「**どうしても避けられないとこにトレーナーが2人いる**」という報告が。パーティが満身創痍のままトレーナーと戦わなければならない状況でレポートに至る
